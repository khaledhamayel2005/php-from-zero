# Hassonah Course Material

The material is organized by topic. Original examples and lecture files were kept, but the folders were renamed and grouped to make browsing easier.

## Main Folders

- `01-course-pdfs/` - all lecture PDF files.
- `02-html/` - HTML chapter examples.
- `03-php-basics/` - PHP syntax, arrays, basic forms, and OOP examples.
- `04-php-forms/` - form handling and form-control examples.
- `05-sessions-and-cookies/` - sessions, cookies, JSON-cookie examples, and login-session examples.
- `06-database/` - MySQL query examples and user-manager database examples.

## Browser Entry

Open:

```text
http://localhost/PHP/Hassonah/
```

or open `index.html` from this folder.
