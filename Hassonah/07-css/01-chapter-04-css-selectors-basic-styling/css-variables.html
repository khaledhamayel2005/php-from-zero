Key Points:

  • Define variables: Use --variable-name: value; in :root
  • Use variables: Use var(--variable-name) anywhere in CSS
  • Benefits: Easy to maintain, change theme colors in one place
  • Scope: Variables can be global (:root) or scoped to specific elements
