  • Form with two two submit buttons

    In this example, we have two submit buttons to add a memory item or to delete it. The form sends the data using post method to process.php, which gets the value from the $_POST array. The submit input type called action its value is retrieved, and then function called based on the action value that sent by the user. The script has two functions: add and delete. The action value retrieved from the $_POST array used as an expression in the switch statement to call the appropriate function.
