# Save Form Data to MySQL

Use this folder to study how form input is collected and inserted into a MySQL table.

## Planned Files

- `index.php` - displays the HTML form.
- `save.php` - validates and stores submitted form data.
- `config.php` - holds the database connection.
- `style.css` - adds the form design.

## File Notes

- Put a comment at the top of each file that says what the file is responsible for.
- Keep validation simple and readable in the first version.
