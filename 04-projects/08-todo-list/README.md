# Todo List

Use this folder for a simple PHP and MySQL todo list project.

## Planned Files

- `index.php` - shows todos and the add form.
- `add.php` - inserts a new todo.
- `delete.php` - removes a todo.
- `config.php` - database connection file.
- `style.css` - todo list styling.

## File Notes

- Put a short English comment at the start of each file.
- Keep the todo flow easy to follow: load, add, delete.
