# Basic CMS Blog

Use this folder for a small CMS/blog project with posts and an admin area.

## Planned Files

- `index.php` - public home page with blog posts.
- `admin/index.php` - admin dashboard.
- `post.php` - single post page.
- `config.php` - database connection and settings.
- `style.css` - blog styling.
