# PHP Projects Roadmap

This folder is organized in playlist order so each project can be studied step by step.

## How To Study Each Project

Every project folder should contain a small starter structure, and every file should start with a short English comment that explains its role.

## Project Order

1. [01-login-registration](01-login-registration) - Login and registration form in PHP and MySQL.
2. [02-crud-operations](02-crud-operations) - Basic CRUD operations with MySQL.
3. [03-save-form-data-to-mysql](03-save-form-data-to-mysql) - Save HTML form data into MySQL using PHP.
4. [04-fetch-display-data-from-mysql](04-fetch-display-data-from-mysql) - Fetch data from MySQL and show it in PHP.
5. [05-basic-cms-blog](05-basic-cms-blog) - Build a basic CMS/blog with PHP and MySQL.
6. [06-contact-form-email](06-contact-form-email) - Contact form and sending email from PHP.
7. [07-upload-images-to-mysql](07-upload-images-to-mysql) - Upload images, store them, and display them.
8. [08-todo-list](08-todo-list) - Simple to-do list with PHP and MySQL.
9. [09-send-email-with-phpmailer](09-send-email-with-phpmailer) - Send email using PHPMailer and Gmail SMTP.

## Common File Roles

- `index.php` - main page and user interface.
- `config.php` - database credentials or app settings.
- `db.php` - reusable database connection.
- `style.css` - page styling.
- `README.md` - short study notes for the project.
- `assets/` - images, icons, uploads, or other support files.

## Study Rule

- Put clear English comments at the top of each file and before each important block.
- Keep the first version small and readable.
- Add the SQL table later in a separate file if the project needs MySQL.

## Next Step

Open any project folder and build the starter files one by one in the same style.
