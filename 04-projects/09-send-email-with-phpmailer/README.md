# Send Email with PHPMailer

Use this folder for sending email with PHPMailer and Gmail SMTP.

## Planned Files

- `index.php` - mail form page.
- `send.php` - sends the email through PHPMailer.
- `config.php` - SMTP credentials and mail settings.
- `vendor/` - Composer packages.
- `style.css` - styling for the mail form.

## File Notes

- Comment the SMTP setup and mail-sending block in English.
- Keep credentials outside the main page logic when possible.
