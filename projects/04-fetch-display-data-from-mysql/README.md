# Fetch and Display Data

Use this folder to learn how to read rows from MySQL and display them in the browser.

## Planned Files

- `index.php` - main listing page.
- `config.php` - database connection file.
- `show.php` - optional page for viewing one record.
- `style.css` - page styling.
