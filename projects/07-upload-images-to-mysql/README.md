# Upload Images to MySQL

Use this folder to learn image upload, file storage, and image display with MySQL.

## Planned Files

- `index.php` - upload form and image list.
- `upload.php` - validates and saves the image.
- `config.php` - database connection and upload settings.
- `uploads/` - stores the uploaded files.
- `style.css` - upload page styling.

## File Notes

- Add English comments for file validation, database save, and display logic.
- Keep upload rules small and clear.
