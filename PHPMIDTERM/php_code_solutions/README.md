# PHP Midterm Code Solutions

This folder contains one organized solution file for each non-duplicate PHP exam/question group extracted from `PHPMIDTERM` and the attached `php code forms M.pdf`.

Duplicate or repeated sources were merged instead of creating extra files. For example, the eStore cart question appears in both `Web_mid_nc2QLqJ_cKt1uav.pdf` and `midTermMarkingSchema_d815FUG.pdf`, so it is solved once in `06_web_mid_estore_cart.php`.

Important: the 11 PHP files are not 11 questions. They are grouped files. After the second audit, they contain 27 PHP/code answer blocks, including repeated short output questions and related HTML forms that submit to PHP scripts.
