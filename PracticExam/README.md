# Practical Exam Training

This folder is a two-day practice track based on the material in `Hassonah`.

Important scope notes:

- Database practice follows `php2.pdf` slides 26-46 only.
- Use PDO: connection constants, try/catch, `query`, `fetch`, `fetchObject`, `prepare`, `bindValue`, `execute`.
- For exam-style password examples, use `md5`, not `password_hash`.
- Keep the code simple enough to write in Notepad.

Suggested sprint:

Day 1:

1. `01-html-form`
2. `02-get-post-validation`
3. `03-arrays-table-search`
4. `04-functions-include`
5. `05-oop-basic`

Day 2:

1. `06-cookies`
2. `07-sessions-login-cart`
3. `08-pdo-database-select`
4. `09-pdo-insert-md5`
5. `10-mock-exam-no-db`
6. `11-mock-exam-with-db`

How to train:

1. Read `question.txt`.
2. Open `starter.php`.
3. Solve it from scratch or complete the TODOs.
4. Compare with `solution.php`.
5. Rewrite the solution once without looking.
